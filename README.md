# DBMS E-Commerce Management System

This project was developed as part of the Database Systems course to demonstrate practical implementation of database design principles in a real-world application. The system is built using React.js for the frontend, ASP.NET Web API for the backend, and MySQL as the database management system.

The project emphasizes database concepts such as Entity Relationship Diagrams (ERD), normalization, relational schemas, primary and foreign key relationships, referential integrity, and CRUD operations. The system allows efficient management of products, categories, users, roles, permissions, and customer orders through a structured and scalable database architecture.
